Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML Sets html codes on selected element. innerText sets just plain texts excluding hidden texts html tags. textContent sets texts within the element regardless of the elements visibility.

Question-3: What is event delegation in the DOM?

Event delegation:Event handling technique where we can listen to child elements by adding eventlistener to their parents.

Question-4: What is event bubbling in the DOM?

Event bubbling: event propagation model ,where an events occurs in child element and it bubbles up to it's parent element and ancestors by three phase.>>
1)capturing Phase: event goes to target element from document.
2)target phase:event occurs at targetted element
3)event bubbles up to ancestors from target element
